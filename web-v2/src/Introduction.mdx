# デジタル庁デザインシステム v2

デジタル庁デザインシステムに準拠した最新のReactコンポーネントライブラリです。

## 技術スタック

- **Vite 6.3+** - 高速なビルドツール
- **React 19** - UIライブラリ
- **TypeScript 5.8+** - 型安全な開発
- **Tailwind CSS v4** - ユーティリティファーストCSS
- **shadcn/ui** - カスタマイズ可能なUIコンポーネント

## デザイントークン

すべてのコンポーネントは、デジタル庁デザインシステムのトークンに基づいています：

### カラーパレット

- **プライマリ**: `#0017C1`
- **セカンダリ**: `#00118F`
- **グレースケール**: 12段階のグレーパレット
- **セマンティックカラー**: エラー、警告、成功、情報

### スペーシング

4pxベースのスペーシングシステム：
- `xs`: 4px
- `sm`: 8px
- `md`: 16px
- `lg`: 24px
- `xl`: 32px

### タイポグラフィ

- **フォントサイズ**: `xs` (12px) から `6xl` (60px)
- **フォントウェイト**: normal (400), medium (500), semibold (600), bold (700)
- **行間**: tight (1.25) から loose (2)

## 使い方

各コンポーネントは、左側のメニューから確認できます。各コンポーネントには：

- インタラクティブなプレビュー
- 設定可能なプロパティ
- 使用例のコード
- ドキュメント

が含まれています。

## コンポーネント

### 基本UIコンポーネント

- **Button** - 様々なバリアントとサイズのボタン
- **Card** - コンテンツを整理するためのカードコンポーネント
- **Alert** - 情報、警告、エラーなどのメッセージ表示

### レイアウトコンポーネント

- **Header** - アプリケーションヘッダー
- **MainLayout** - 基本的なページレイアウト

## アクセシビリティ

すべてのコンポーネントは、WCAG 2.1 AA準拠を目指して実装されています：

- 適切なARIA属性
- キーボードナビゲーション対応
- スクリーンリーダー対応
- 十分なカラーコントラスト
